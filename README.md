# 🏋️ Academia Bot - Sistema de Atendimento Inteligente

Sistema completo de atendimento para academias, focado em captação de leads, automação de mensagens e organização comercial.

## 🚀 Funcionalidades

- Atendimento automático inicial (bot)
- Identificação do interesse do cliente (modalidades, planos, etc)
- Criação automática de leads
- Painel de atendimento centralizado
- Histórico completo de conversas
- Resposta manual do atendente
- Controle de status (novo_lead, em_atendimento, atendido)

## 💡 Objetivo

Evitar perda de clientes, organizar o atendimento e melhorar a conversão de leads em alunos.

## 🛠️ Tecnologias

- Node.js
- Express
- SQLite
- HTML / CSS / JS (frontend simples)

```bash
npm install
node src/server.js
